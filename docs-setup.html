Docs SetUp


First Steps

The general first steps:

  • If it's not a private intranet site, remove the meta-robots-nonindex line from all html files. Than remove the disallow back-slash in the robots.txt file
  • Change the css and js lines in the html head of every html file if you are using CDNs to load those things.
  • Modify the href links to suit your site's top-down folder structure.

CDN URLs

The CDN urls to use to load resoruce dependencies from the cloud:

Roboto Font

<link href="https://fonts.googleapis.com/css?family=Roboto+Slab:100,300,400,700&subset=cyrillic,cyrillic-ext,greek,greek-ext,latin-ext,vietnamese" rel="stylesheet">
